## 前置要求

- Linux 系统
- root 权限（需要 sudo 运行）
- 已安装 Docker 和 Docker Compose（安装器会自动检测）

## 安装 FlecBlog

```bash
curl -fsSL https://install.flec.top | sudo bash
```

安装器会依次引导你完成以下步骤：

### 1. 选择部署方式

```text
  部署方式:
    [1] Docker 部署 (推荐)
    [2] 裸机部署 (systemd + Nginx) [暂不可用]

  请选择 [1]:
```

目前仅支持 Docker 部署方式。

### 2. 环境检测

安装器会自动检测 Docker 和 Docker Compose 的安装状态和版本。如果未安装，会提示你安装 Docker。

### 3. 配置端口

```text
  端口配置（直接回车使用默认值）：
  Server 端口 [8080]:
  Blog 端口 [3000]:
  Admin 端口 [4000]:
```

如果端口已被占用，安装器会提示冲突并让你重新设置。

### 4. 确认安装

安装器会展示配置摘要，确认后开始部署：

```text
  配置确认:
    部署方式: Docker 部署
    安装路径: /opt/flecblog
    端口: Server=8080, Blog=3000, Admin=4000

  确认安装? [Y/n]:
```

### 5. 自动部署

安装器会自动完成：

- 创建目录结构
- 生成 `.env` 和 `docker-compose.yml` 配置文件（数据库密码和 JWT 密钥自动生成）
- 拉取 Docker 镜像
- 启动服务
- 等待服务就绪

### 6. 访问服务

部署完成后会显示访问地址：

- 博客端：`http://服务器IP:3000`
- 管理端：`http://服务器IP:4000`

> **初始账号**
> 初始账号密码为 `admin@example.com` / `123456`

## 其他命令

### 检查更新

```bash
sudo flecb upgrade --check
```

### 升级版本

```bash
sudo flecb upgrade
```

### 修改配置

修改已安装实例的端口、API 地址等配置：

```bash
# 交互模式
sudo flecb reconfig

# 指定参数
sudo flecb reconfig --server-port 8081 --blog-port 3001
```

### 备份数据

```bash
# 创建备份
sudo flecb backup

# 指定输出路径
sudo flecb backup --output /tmp/my-backup.tar.gz

# 从备份恢复
sudo flecb backup --restore /tmp/my-backup.tar.gz
```

### 卸载

```bash
# 卸载但保留数据
sudo flecb uninstall

# 彻底删除包括数据
sudo flecb uninstall --purge
```

### 查看版本

```bash
flecb --version
```

## 配置文件位置

安装完成后，配置文件保存在以下位置：

| 文件             | 路径                                 | 说明                  |
| -------------- | ---------------------------------- | ------------------- |
| 安装器            | `/usr/local/bin/flecb`             | 安装器可执行文件            |
| 安装器配置          | `~/.flecblog/config.json`          | 安装信息、端口、版本等         |
| Docker Compose | `/opt/flecblog/docker-compose.yml` | 容器编排配置              |
| 环境变量           | `/opt/flecblog/.env`               | 数据库密码、JWT 密钥、API 地址 |
| 数据目录           | `/opt/flecblog/data/`              | 上传文件等持久化数据          |
| 数据库目录          | `/opt/flecblog/postgres_data/`     | PostgreSQL 数据       |
