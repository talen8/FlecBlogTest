## 部署相关

### 页面可以打开但没有数据？

检查 `API_URL` 配置是否正确，确保博客端能访问到后端 API。

### 端口被占用？

修改 `docker-compose.yml` 中的端口映射：

```yaml
ports:
  - "3001:3000"
```

### 配置修改后没有生效？

```bash
docker-compose up -d --force-recreate
```

## 功能相关

### 如何修改管理员密码？

登录管理后台后，在「系统管理 → 用户管理」中找到对应账号，点击编辑即可修改密码。

### QQ 登录无法使用？

QQ 登录需要等待应用审核完成后才可使用。审核完成之前，博客端的 QQ 登录按钮需要保留（在管理端启用 QQ 登录方式）。如需加速审核，可在 QQ 互联平台联系客服。

### 飞书通知和邮件通知可以同时使用吗？

可以。两者是独立配置的，你可以同时启用，或只启用其中一种。

### 如何自定义表情包？

前往 <https://github.com/talen8/emojis> Fork 该仓库进行自定义修改，然后在博客配置中将表情包 URL 替换为 `https://fastly.jsdelivr.net/gh/<用户名>/<仓库名>/emojis.json`。

### 如何备份数据？

1. **数据库备份**：使用 `pg_dump` 或宝塔面板的数据库备份功能
2. **文件备份**：备份 `/srv/flecblog` 目录下的上传文件
3. **配置备份**：导出 `.env` 和 `docker-compose.yml`
